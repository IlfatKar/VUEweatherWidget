# [VUE] Weather Widget

A widget that displays the weather in the specified cities.

## How to build

```
$ npm i
$ npm run build
```

## How to use

```html
<weather-widget />
<script
  type="text/javascript"
  src="https://cdn.jsdelivr.net/gh/IlfatKar/VUEweatherWidgetPreview/app.js"
></script>
```
